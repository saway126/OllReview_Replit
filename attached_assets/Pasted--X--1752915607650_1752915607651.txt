[프리미엄 체험단 및 기자단]
> 최적화블로그 섭외 기능
> 자동입찰 및 결제 기능
> 상위노출 원고 생성기능 (보장X)
> 자동 스댓공 기능
> 광고주와 카테고리별 최블 다이렉트 매칭 사이트
> 별첨 : 최적화 블로그 육성 기능
> 추후 다른 SNS플랫폼 확장 가능 (글, 영상, 카드뉴스등)

1. 최적화블로그 섭외 기능
1) 블로그연구소에서 키워드별 최적화 블로그 아이디 분류
2) 해당 아이디를 우리 프로그램에 삽입
3) 메일을 보낼 수 있는 비실명 아이디 구비
4) 핸드폰 테더링으로 IP값 변경
5) 해당 아이디로 로그인해서 추출한 최적화 블로그 50개씩 홍보성 메일 전달

2. 기자단 플랫폼 구현
: 광고주모드, 인플루언서모드, 관리자보드 3개로 구성
[수정사항1]
1. 관리자 대시보드
1) 관리자로 로그인하면 에러가 뜨는 상황이야. 확인하고 수정해줘.

2. 광고주 대시보드
1) 타겟설정에서 연령대를 중복선택할 수 있게 확인 후 수정
2) 타깃 세그먼트 설정 삭제
3) 타깃 필터에서 연령대, 지역, 관심사 중복 선택 가능하게 수정
4) 관심사는 네이버 쇼핑 대카테고리 기준으로 다양하게 설정 가능하게 수정
6) 일일예산을 정하면 관리자가 정한 건별 예산에 따라 일수량이 자동으로 설정되게 변경
7) 캠페인 생성 에러 확인 후 수정
8) QR스캔수와 전환수는 새로 세팅한 자사몰의 각 상품 링크에 픽셀 심어두고 자동으로 카운팅 및 연동되게 세팅
9) 지금 왼쪽 카테고리에 캠페인, 분석, 결제를 누르면 404found 에러가 뜨는 상황이야. 분석 후 수정해줘.

3. 파트너 대시보드
1) 파트너 대시보드에서는 관리자가 지정해준 광고주의 상품을 확인하고 발송한 데이터를 등록할 수 있게 만들어줘.
2) 샘플업체, 발송계획(시작일자, 수량), 세그먼트(발송데이터등록), 수익리포트로 카테고리를 나누고 현재 오류나는 부분 확인하고 수정해줘.
3) 발송계획 버튼이 안눌리는 상황이야. 확인하고 수정해줘.


[수정사항2]
0. 전체적인 수정사항
1) 체험단, 기자단 사이트처럼 전면 재구성한다.
2) 광고주는 캠페인을 생성하고 모집기간 동안 파트너를 모집할 수 있다.
3) 파트너들은 원하는 캠페인을 신청할 수 있다.
4) 광고주들은 모집기간 내에 파트너를 선정해야한다.
5) 선정된 파트너들은 발송 시작전까지 광고주들에게 샘플이나 할인쿠폰 전단지를 받는다.
6) 파트너는 발송이 시작되면 그날그날 홍보물이 동봉된 운송장을 입력한다.
7) 관리자는 홍보물의 QR코드 트래픽을 분석해서 사이트에 등록하여 광고주들이 이전 날 데이터를 볼 수 있게 한다.
8) 해당 파트너사가 배송했던 홍보물의 카테고리별 도달율을 기록하여 남기고 광고주들이 새로운 캠페인을 생성하고 파트너 선정시 도움이 되도록 기록에 남긴다. 

1. 관리자 대시보드
1) 전체적인 관리 및 정산할 수 있게 전면 수정

2. 광고주 대시보드
1) 캠페인 생성에서 타깃세그먼트설정에서 CSV파일 업로드나 조건필터란은 불필요하니 삭제해줘.
2) 왼쪽 카테고리에서 캠페인, 분석, 결제가 같은걸 보여주고 있어. 확인하고 수정해줘.


3. 파트너 대시보드
1) "파트너 : 업체명" 이렇게 표시되게 변경해줘.
2) +발송계획추가는 아무런 의미 없는 것 같아 삭제해줘.
3) 샘플업체 탭에서는 관리자가 지정한 업체를 승인, 거절할 수 있는 탭을 만들어주고 승인하면 진행중, 대기중인지 표시되게 수정해줘.
4) 발송계획 탭에서는 해당 파트너사가 연관있는 샘플 카테고리 신청란을 추가해줘. 즉, 파트너사가 원하는, 본사와 가장 적합한 샘플 카테고리와 수량 계획을 표시하는 곳으로 해주고 이 부분을 관리자 대시보드에서 파트너별 정보를 볼 수 있게 해줘.
5) 세그먼트는 발송등록으로 변경해줘. 발송데이터등록란이 2개나 있어서 하나는 삭제해줘. 단일 등록이나 엑셀 포맷으로 대량등록할 수 있게 추가해줘. 엑셀 샘플 다운 받는 곳 추가해줘.(캠페인선택, 발송날짜, 운송장번호, 상품명, 메모) 
6) 수익리포트는 달력으로 선택해서 날짜별로 볼 수 있게 해주고 실제 종합몰에 설치된 픽셀값 수정해줘.
